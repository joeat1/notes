# 自然语言处理 #


> 对于人来说，语言相关的基本技能为“听、说、读、写”，而其中听和读属于信息输入，说和写属于信息输出。而对计算机而言，自然语言向计算机语言迁移，需要处理自然语言理解**（NLU）**和自然语言生成**（NLG）**两个任务。生成和理解共同使用相同的知识。当前，计算机的自然语言处理功能可以作为人的理解和写作的助理来使用，此时需要向读者和作者的方向发展。
>
> <u>多媒体信息</u>如何理解和生成也是在信息载体出现大量丰富后需要考虑的问题


## 挑战

+ 人类语言具有非结构化，歧义性，多样性的特点，难以形式化描述
+ 人类语言一直在进化，新词组、新表达等形式不断产生
+ 自然语言的离散性，组合性和稀疏性
+ 如何使得机器像人一样善于产生和理解语言，具有表达，感知，理解复杂且微妙信息的能力

## 人是如何理解和表达的？

+ 从过去的实例迁移到新的内容
+ 大脑正确的表示和映射信息

## 自然语言理解（NLU）

+ 文本含义
  + 语义     词义排歧义（WSD）、词组合
  + 篇章     指代消解  结构标注
  + 语用    上下文
+ 文档摘要
+ 关系抽取（RE）
+ 情感分析         条件随机场（CRF）



## 自然语言生成（NLG）   

> 适用于说明书的自动生成，百科全书的描述等
> 核心任务：文档规划（内容确定，结构分组），微观规划（词汇选择，所指表达，结构映射）和表层实现（语言实现）等  算法和表达方式

+ 适用环境
+ 需求分析
+ 评估方式
+ 相关问题
+ 体系结构

输入 主谓宾，时态和语法特点，根据格式规则，输出句子。



心理模型和形式模型
